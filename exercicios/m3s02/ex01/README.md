## Objetivo

Criar um codigo em Python que separe:

- `X`: variaveis preditoras
- `y`: variavel resposta

- [separar_xy.py](/C:/Users/deni_/projeto_analise_preditiva/exercicios/m3s02/ex01/separar_xy.py)


`investimento_marketing`, `preco_medicamento` e `mes`, e separa `y` como
`demanda`.

Com a base atual:

- `y` -> `(20000,)`
